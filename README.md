# GitHub User Dashboard

GitHub User Dashboard is a React + Vite web app that lets users search for GitHub profiles and view basic user information through the GitHub API.


## Features

- Search for a GitHub user by username.
- Display the user's information when the account exists.
- Show an error message when the username does not exist.
- Built with React and Vite for fast development and a lightweight setup.


## Technologies

- React
- Vite
- GitHub API


## API

This app uses the GitHub API to fetch user data.
